# 桌面客户端

## 1. 概述

OmniDesk 桌面客户端是"三部分架构"（浏览器端 + 桌面端 + 通知服务）中的桌面应用层，基于 **PyQt5** 构建，兼容 Python 3.8 和 Windows 7。

## 2. 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| GUI | PyQt5 | 替代 PyQt6，兼容 Win7 |
| Python | 3.8+ | 最低兼容版本 |
| 打包 | PyInstaller | 生成独立可执行文件 |

## 3. 功能

| 功能 | 说明 |
|------|------|
| 内嵌浏览器 | 加载 Web 界面 |
| 通知中心 | 角标 + 列表 + 标记已读 + 系统托盘 |
| 系统托盘 | 右键菜单：打开/设置/退出 |
| 离线缓存 | 网络断开时缓存数据 |
| 健康检查 | 定期检查后端连接状态 |
| 一键打开 Web | 托盘菜单跳转浏览器 |

## 4. 构建与分发

```bash
pyinstaller desktop.spec
```

CI 自动构建并上传 GitHub Release。离线部署脚本提供 `install-desktop` 子命令。
