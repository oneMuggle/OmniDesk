# 排班管理使用指南

## 1. 查看排班表

进入"排班管理"页面查看排班信息。

## 2. 创建排班

点击"新建排班"，填写日期、班次、值班人员、值班领导。修改后系统自动通知相关人员。

## 3. 试验管理

在"试验管理"页面查看和创建试验记录。

## 4. 申请换班（v0.5.0+）

适用场景：发现未来某天自己值班，但当天有事，想与某位同事对调。

### 4.1 提交换班申请

1. 进入 **我的工作台 → 我的排班**（`/api/events/my-schedule/`）
2. 找到要换班的日期 → 点击 **"申请换班"**
3. 选择接收方同事 + 填写换班理由 → 提交

系统会自动通知接收方 + HR。

### 4.2 等待审批

- 状态：申请方页面可见"待审批 / 已批准 / 已拒绝 / 已过期"
- 未处理请求 48 小时后自动过期 → 通知 HR + 申请方

### 4.3 HR 审批视角

HR 进入 **排班管理 → 换班审批列表**，对每条申请：
- **批准**：系统自动改原 Schedule，同时通知双方
- **拒绝**：通知申请方，原因记入审计

### 4.4 撤销

申请人在 HR 审批前可主动撤销，状态置为"已取消"。

### 4.5 字段级权限

- 申请人能看到自己提的全部申请
- 接收方能看到与自己相关的全部申请
- HR 看到所有
- 其他用户看不到他人申请（防隐私泄漏）
