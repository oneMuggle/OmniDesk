# 智能助手使用指南

> 📅 **最近更新:2026-06-06** — 已与最新模块功能同步,涵盖 12 类工具查询、知识库管理、对话历史与审计面板。

## 1. 进入智能助手

在左侧菜单中找到 **"AI 助手 → 智能助手"**,点击进入聊天界面。

聊天界面支持:
- **流式打字效果**:AI 回答逐字显示,无需等待完整生成
- **多轮对话**:自动保存上下文,后续提问可引用前文
- **Markdown 渲染**:回答中的代码、表格、列表自动格式化

## 2. 你能问什么

| "下周三谁负责?" | 指定日期排班查询 |
| "帮我看下李明的班次" | 按姓名查询 |

### 2.3 文档与公文

### 2.4 备忘录

### 2.6 新闻与公告

| 示例问题 | 说明 |
|----------|------|
| "本周公司新闻" | 新闻列表 |
| "通知:放假安排" | 公告搜索 |

### 2.6.1 公司公告(`announcement_query`,阶段 3 新增)

通过 `communication.Post` 表查询已发布、未过期、未归档的公告(最多 10 条最新)。

**典型问题**:
- "这周有什么公告"
- "本周例会通知"
- "最近的安全检查通知"
- "公司放假安排"

**返回**: 公告卡片,显示标题(过期标橙色 tag)、发布人、发布日期、内容(200 字截断 + `...`)。

### 2.6.2 合规问题(`compliance_query`,阶段 3 新增)

通过 `compliance.ComplianceIssue` 表查询待处理/处理中的合规问题,按业务优先级排序(紧急 > 高 > 中 > 低)。

**典型问题**:
- "待整改"
- "张三还有几条待整改"
- "紧急的合规问题"
- "即将到期的整改"
- "签字相关的问题"

**返回**: 合规卡片,显示问题类型 + 严重程度 tag(红/火山/橙/灰)+ 状态 tag(已解决=绿/进行中=蓝)、所属项目、截止日期、问题位置、描述。

### 2.6.3 内网外链(`external_link_query`,阶段 3 新增)

通过 `external_integration.ExternalLink` 表查询公司内网工具导航(VPN、Jira、Confluence 等)。

**典型问题**:
- "公司 VPN 怎么登录"
- "Jira 怎么打开"
- "所有内网工具"
- "研发相关链接"
- "SSO 登录地址"

**返回**: 外链卡片,显示名称(可点击,SSO 启用时跳 SSO 端点)+ 紫色 SSO 标签、分类、地址、说明。

### 2.9 传感器

| 示例问题 | 说明 |
|----------|------|
| "设备 DEV-001 当前状态" | 设备状态 |
| "有告警的传感器" | 告警列表 |

### 2.10 知识库问答

| 示例问题 | 说明 |
|----------|------|
| "公司的请假流程是什么?" | 知识库文档问答 |
| "VPN 怎么连?" | 操作指引 |

> 💡 **知识库问答**会显示引用来源,可点击查看原文。

### 2.11 多工具复合查询

智能助手支持**一次提问触发多个工具**,例如:

- "张三今天的排班和他最近的项目"→ 人员 + 排班 + 项目
- "本周公告和合规检查计划"→ 公告 + 合规

回答时会合并多个工具的结果,标注数据来源。

### 2.12 不支持的内容

- ❌ 文献搜索(CNKI / IEEE / PubMed)— 暂无授权,见 [技术手册 16 §6 Phase 9](../technical/16-smart-assistant.md)
- ❌ 实时网页搜索
- ❌ 图片/语音输入(P3 阶段评估)

## 3. 对话管理

### 3.1 新建会话

- 点击聊天界面左上角的 **"+"** 按钮创建新会话
- 新会话默认标题为前 50 字,可手动修改

### 3.2 切换会话

- 左侧侧边栏列出所有历史会话,按最近时间倒序
- 点击任意会话即可恢复上下文继续对话
- 搜索框支持按标题/内容关键词过滤

### 3.3 删除会话

- 在侧边栏右键点击会话 → 删除
- 或在会话详情页右上角点击删除按钮

### 3.4 反馈与重试

每条 AI 回答下方提供:
- 👍 点赞
- 👎 点踩(自动收集到训练反馈)
- 📋 复制
- 🔄 重试(相同问题重新生成)

## 4. 知识库管理

进入 **"AI 助手 → 知识库管理"** 页面。

### 4.1 上传文档

1. 点击 **"上传"** 按钮
2. 选择文件(支持 PDF / Word / Markdown / TXT)
3. 填写标题(必填)、分类(可选)、标签(可选)
4. 点击确认,文档进入"待解析"状态

### 4.2 状态流转

| 状态 | 含义 |
|------|------|
| 待解析 | 等待 Celery worker 处理 |
| 解析中 | Ragflow 正在向量化 |
| 已完成 | 可被 RAG 工具检索 |
| 失败 | 查看错误日志后可重试 |

### 4.3 删除文档

- 列表中点击删除 → 二次确认
- 删除时同步取消 Celery 任务与 Ragflow 中的索引

> ⚠️ 删除操作不可恢复,删除前请确认不再被引用。

## 5. 审计面板(管理员)

进入 **"AI 助手 → 审计面板"**(需管理员权限)。

可查看:
- **总览卡片**:今日对话数、Token 消耗、费用估算、平均响应时间
- **意图分布**:饼图展示用户问题类型
- **工具成功率**:柱状图(各工具调用成功率)
- **每日用量趋势**:折线图
- **异常告警列表**:超时 / 失败 / 高额 Token
- **详情查询**:可按用户/时间/意图/关键词过滤

## 6. 统计页面

进入 **"AI 助手 → 统计"**(需管理员权限)。

展示:
- 个人用量(7/30 天 token、对话次数)
- 工具使用偏好
- 反馈统计(点赞/点踩比)

## 7. 常见问题

### Q1:提问后没反应?

- 检查浏览器 Network 面板,看 `/api/smart-assistant/chat/stream/` 是否正常返回
- 流式响应首帧需 1-3 秒,属正常(可联系管理员开启 TTFT 优化)

### Q2:回答不准确?

- 试用"换一种方式提问"
- 或直接联系管理员,管理员可在审计面板查看 LLM 完整 prompt 与 tool 输出,定位问题

### Q3:提示"请求过于频繁"?

- 单用户每分钟最多 30 次请求(系统限流)
- 等待 1 分钟后重试,或联系管理员调高限制(不推荐)

### Q4:工具没找到我要的信息?

- 工具返回 `found=False` 时,助手会尝试通用回答
- 通用回答可能不准确,建议:
  - 调整提问关键词
  - 检查知识库是否已上传相关文档
  - 联系管理员添加/调整工具

### Q5:对话历史丢失?

- 默认保留 30 天(可在系统设置调整)
- 删除会话不可恢复

### Q6:支持多语言吗?

- 当前主要为中文问答,英文支持有限
- 多语言切换在 P3 阶段评估
