# 排班日历修正计划

## 问题

试验日程和排班日程页面的日历显示的不是当前月份。

## 根源

在 `omni_desk_frontend/src/components/BaseSchedule.jsx` 文件中，`initialDate` 被硬编码为 `"2025-04-01"`，导致日历无法显示当前月份。

## 修复方案

修改 `omni_desk_frontend/src/components/BaseSchedule.jsx` 文件，移除 `initialDate="2025-04-01"` 这一行。这样 `FullCalendar` 组件在没有 `initialDate` 属性时，将自动显示当前月份。
