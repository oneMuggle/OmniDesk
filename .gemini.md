
# Calendar Project

This project is a full-stack calendar application, consisting of a React-based frontend and a Django-based backend.

## Sub-projects

- **`calendar_with_react/`**: The frontend application built with React. See the `.gemini.md` file in that directory for more details.
- **`DRFForVue/`**: The backend API built with Django REST Framework. See the `.gemini.md` file in that directory for more details.

For more information about each sub-project, please refer to the `.gemini.md` file within its respective directory.
