
# omni_desk_backend

## Project Overview

This is a Django-based backend for a calendar application. It provides a RESTful API for managing events, users, and other resources. The project uses Django REST Framework to build the API endpoints.

## Key Technologies

- **Framework:** Django
- **API:** Django REST Framework
- **Authentication:** djangorestframework-simplejwt
- **Database:** PostgreSQL (implied by psycopg2)
- **CORS:** django-cors-headers

## Development

To run the project in development mode, first install the dependencies:

```bash
pip install -r requirements.txt
```

Then, run the development server:

```bash
python manage.py runserver
```

This will start the development server, typically at `http://localhost:8000`.
