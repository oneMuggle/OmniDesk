# 权限系统用户手册

## 1. 权限类型

### 1.1 角色权限
- 管理员: 系统最高权限
- 经理: 部门管理权限
- 普通用户: 基础权限

### 1.2 功能权限
- 用户管理: manage_users
- 日程管理: manage_schedules
- 设备管理: manage_equipment

## 2. 权限申请流程

1. 登录系统
2. 进入"权限申请"页面
3. 选择需要申请的权限
4. 填写申请理由
5. 提交申请
6. 等待管理员审批

## 3. 权限管理

### 3.1 管理员操作
1. 进入用户管理界面
2. 选择目标用户
3. 分配/撤销权限
4. 保存更改

### 3.2 查看当前权限
1. 点击个人头像
2. 选择"我的权限"
3. 查看权限列表

## 4. 常见问题

### 4.1 为什么无法访问某些功能？
- 检查是否具有相应权限
- 联系管理员申请权限

### 4.2 权限申请被拒绝怎么办？
- 查看拒绝理由
- 补充申请材料重新提交
- 联系管理员沟通
